Troubleshooting

A collection of KQL I use to troubleshoot incidents in Sentinel.

SentinelHealth
| where SentinelResourceName in ("Successful logon from IP and failure from a different IP")
| where Status == "Failure"
| project TimeGenerated, SentinelResourceName, Status
| sort by TimeGenerated desc

SentinelHealth Selects data from the SentinelHealth table.
| where SentinelResourceName in (“Successful logon from IP and failure from a different IP”) Filters records to include only those with specific resource names related to logon success and failure.
| where Status == “Failure” Further filters the data to include only failed login attempts.
| project TimeGenerated, SentinelResourceName, Status Selects only the columns for the time of the event, resource name, and status.
| sort by TimeGenerated desc Orders the results by the most recent events first.
